第三届 超级智能体 Campathon

Super Agent Campathon

本次大赛聚焦“端侧智能体”，旨在解决AI应用中网络延迟高、API成本昂贵及用户数据隐私泄露的挑战，共同定义有“灵魂”的 AI 及下一代交互。

比赛结束，感谢参与（点击查看报名网站）

迈向终极 AIOS 的 Camp + Hackathon

“超级智能体大赛”是 GOSIM 大会的年度开发者核心活动。我们首创 “Campathon” 赛制 (训练营+黑客松)，让你在学习后立即创造。

第一步：系统学习 (Camp)

在为期两周的线上技术训练营中，开发者将学习并掌握以MoFA为核心的模块化开发框架。该框架专为端侧设备设计，旨在实现低延迟、保护用户隐私、并减少长期运行成本的对话体验。

第二步：动手创造 (Hackathon)

完成训练营后，开发者将在黑客松环节动手实践，将自己的创意构建成一个实用、个性化的智能体应用。我们的目标是产出一批能解决真实世界问题的高质量、可运行的端侧智能体项目。

最终目标：定义未来

我们期待看到你的作品，共同探索人机交互的下一步发展。

创造一个有“灵魂”的智能体

你将构建一个鲜活的、仿佛拥有自主意识的智能体。你的核心挑战是实现极致的实时性、自然的交互感、丰富的情感力与强大的扩展性。

你可以探索的方向包括但不限于：

  • 🧠 高效的日程助理：一个能离线使用、高效处理日程的个人助理。
  • 👨‍🏫 能离线使用的学习工具：进行外语陪练、解答专业问题、激发学习兴趣。
  • ❤️ 懂你的情感伴侣：倾听你的烦恼，分享你的喜悦，提供温暖的情绪支持。
  • 👓 智能硬件的“大脑”：为智能眼镜、智能音箱注入强大的对话能力，实现真正的“免提”操作。

Sky's the Limit！

为何参赛？

🏆 顶级大奖与荣誉

赢取包括 RTX 5090 笔记本电脑、MacBook Pro 在内的顶级硬件大奖。前6名队伍将获邀参加 GOSIM Hangzhou 2025 现场决赛，我们将为每队报销一位成员的差旅并赠送大会门票与礼品。*注：所有奖励均以团队为单位（一队一份）。*

🚀 前沿端侧技术实践

我们提供以MoFA为核心的模块化开发框架来降低开发门槛，让你亲手体验最前沿的端侧AI Agent技术。

🧑‍💻 核心导师全程指导

来自核心研发团队的专家将提供线上技术训练营与指导，助你扫清障碍，实现从入门到精通。

技术武器库

Moly LLM 前端 App: 一个为大语言模型打造的前端交互应用。
MCP 开放计算协议: 连接海量第三方服务（如RESTful API），让你的智能体无所不能。
MoFA 框架: 专为Agent打造的高效、可扩展的模块化开发框架，管理生命周期。
DORA 数据流引擎: 用于设计和实现 Agent 的内部逻辑与复杂任务流程，支持线性和非线性任务流。
MoFA Stage 开发平台: 一站式“智能体IDE”，提供开发、断点调试、性能分析和部署的全方位支持。
AIOS 核心框架: 你的智能体在端侧设备上高效运行的心脏。
